# BioLM Community

Welcome to the **BioLM Community** GitHub organization.

This org is a home for community-driven projects and collaborations around [**BioLM**](https://biolm.ai/).

If you're a developer, researcher, or student interested in biological foundation models, this is where you can contribute and build alongside the BioLM team and community.

---

## What kinds of projects live here?

Some example project types:

- Develop a new **Protocol**
- **Lab validate** a Protocol
- Benchmark a **model family** on a task
- Author a community **tutorial / guide / notebook**
- Develop a BioLM-powered **bioinformatics workflow** (Nextflow, Galaxy, Kedro, etc.)
- Develop a new **community model**

Projects are usually posted as GitHub Issues with scope, expected outputs, and difficulty level.

---

## Outputs we value

Some example outputs (roughly ordered by impact):

- Published papers
- Published contributions that we can feature on the platform
- White papers and co-marketing materials
- GitHub repos and open-source tools
- Blog posts and social posts

---

## How to get started

1. Join the [BioLM Community Slack](https://biolm.ai/community/)
2. Browse open Issues / posted projects
3. Comment on projects you're interested in contributing on
4. Ask questions early
5. Submit your work via PR or as instructed
